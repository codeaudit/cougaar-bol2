BOL Configuration

Provides an overview of Cougaar configuration files. The configuration files covered in this tutorial are the *.ini format. Cougaar is switching to the support of configuration files in XML format, these will be covered in a future trail. The final lesson serves as a guide to configuring the BOL society.

  Agent Configuration Basics
  Using the Prototype-ini.dat configuration file
  LDM XML, creating assets and prototypes from XML files
  Using the q.xml (query file) to configure SQL databases and queries
  Setting up the base BOL society
